Welcome to Coh-Metrix-Dementia, a free text analysis tool for dementia classification and rating.

Coh-Metrix-Dementia is a text analysis tool intended as an aid to the identification and quantification of dementia. The current version of the tool targets Alzheimer's Disease and Mild Cognitive Impairment.
